# bibook-pug-sass-jquery-vitejs

Landing page for Bibook website using Pug, Sass, jQuery... and ViteJS as a build tool 

# Initial run:

* Install Node.js
* `npm install`
* `npm run dev`
* `npm run build`
* `npm run preview`
